# 🏄 Personal Website

## 📚 Stack:
- Next.js
- React
- Editor.js
- SCSS
- Fullpage.js

## 🪄 Features:
- Multipage UI
- Individual projects with in-depth description
- Facts about me
- My contacts

## 🛠️ Setup Instructions:
- Clone this repository
- `npm ci` to install all dependencies 
- `npm run dev` to run in dev environment
